User Agent
▲
▼
Browser, version and os tests are to show how trivial it is to detect that you are using Firefox, the version and your os. However, some have real entropy, such as the math hashes used or the pixel values per zoom setting, and returning the browser/os here is just for fun. Firefox includes the Tor Browser. Detecting Tor Browser as a subset is a bonus: there are other ways to detect Tor. Each test is independent of the others.
